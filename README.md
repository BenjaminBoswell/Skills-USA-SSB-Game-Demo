# Skills-USA-SSB-Game-Demo
This is the game demo for the skills USA competition we will be making the SSB First level prototype of 2026 this game will be an example of a 3D top down shooter where you play as 4 tradesman who one day got teleported into the future.

To keep this project organized Benjamin Boswell will be the main director of this game, the development of this on his end will be concept art and models and animations, the second lead will be Apollo he will be the lead programmer, game mechanic creator and asset implementations director, all other developers will focus on making the game demo come together.

We will focus on making the first level with a modular system of assets and code there fore we can add on and implement further when need be. The first levels assets must be modular, and same with the players mechanics and some enemies behavoirs. We also need to make it optimized and not take to much storage space and out main philosiphy will be KISS (Keep It Simple Stupid) Lets not overly complicate this game project. GO TEAM!!!
